🐍 SNAKEY! — A Colorful Snake Game
SNAKEY! is a fun and visually vibrant browser-based Snake game built with pure HTML5, CSS, and JavaScript — no libraries or frameworks needed.

Guide your snake around the playfield to eat colorful fruits 🍊🍓🍋🍇🥝, grow longer with every bite, and survive as long as possible without hitting the walls or your own tail!

🎮 Features
Progressive Difficulty — The snake speeds up as your score increases, keeping the challenge alive across multiple levels.
Colorful Fruit System — 5 unique fruit types (orange, strawberry, lemon, grape, kiwi), each drawn with canvas-rendered detail including highlights, stems, and leaves.
Particle Effects — Satisfying burst animations play every time you eat a fruit.
Screen Shake — A short screen-shake effect triggers on death for extra impact.
Animated Food — Fruits bob up and down with a pulsing scale animation to catch your eye.
High Score Persistence — Your best score is saved locally using localStorage, so it persists between sessions.
Pause & Resume — Pause at any time using the P key or the on-screen button.
Sound Effects — Synthesized audio feedback for eating and game over using the Web Audio API.
Polished UI — A clean HUD showing Score, Best, and Level, with smooth overlay transitions for Start, Pause, and Game Over screens.
🕹️ Controls
Key	Action
W / ↑	Move Up
A / ←	Move Left
S / ↓	Move Down
D / →	Move Right
P	Pause / Resume
R	Restart (after Game Over)
🛠️ Tech Stack
HTML5 Canvas — All game rendering (snake, food, particles, animations)
Vanilla CSS — Custom design system with a warm cream-and-lime color palette
Vanilla JavaScript — Game loop, collision detection, input handling, audio synthesis
Web Audio API — Procedurally generated sound effects
localStorage — Persistent high score storage
Google Fonts — Fredoka One and Nunito for the playful UI typography
Best part is no installation or setup needed!
Clone or download the repository.
Open snake-game.html directly in any modern web browser.
Click ▶ Play Game and start eating!
